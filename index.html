The Apache Maven SCM Provider Impl is based on svnkit and so use Java process to run svn subversion operations.

<dependency>
  <groupId>com.google.code.maven-scm-provider-svnjava</groupId>
  <artifactId>maven-scm-provider-svnjava</artifactId>
  <version>2.1.5-SNAPSHOT</version>
</dependency>

Requirements: JDK 1.8+, Maven 3.6.3+

git clone https://github.com/olamy/maven-scm-provider-svnjava.git
cd maven-scm-provider-svnjava
mvn install
